# ⛳ GolfIQ

GolfIQ is a modern golf score tracking app built with .NET MAUI.

Track your rounds, monitor your performance, and view key statistics all from your mobile device.

## Features

### 🏌️ Round Tracking
- Add new golf rounds
- Record course name
- Record date played
- Record score
- View recent rounds

### 📊 Statistics
- Average score
- Best score
- Worst score
- Total rounds played
- Performance insights

### ⚙️ Settings
- Player preferences
- Data management
- Future customization options

## Built With

- .NET MAUI
- C#
- XAML
- SQLite (Local Storage)


2. Open in Visual Studio 2022+

3. Restore NuGet packages

4. Run on Android or iOS
